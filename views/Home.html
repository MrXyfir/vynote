Vynote

Vynote is an easy and productive way to store and organize your ideas, code, snippets, and lists.

Features

  • Free to Use: Simply sign in with your Xyfir Account to use Vynote
  • Folder System: Keep your documents organized in nestable folders
  • Multi-Tasking: You can have multiple documents open in different tabs for easy multi-tasking
  • Multiple Document Types: Use the right tool for the job. See Document Types
  • Document Collaboration: Invite other Vynote users to work on a document with you
  • In the Cloud: Your documents are automatically saved in the cloud. This means you can access your documents from all of your devices!

Document Types

Page

Page documents are a great multi-purpose document solution. Page documents support the full Markdown Syntax and offer both an edit and HTML-preview modes. Use them for simple plain-text notes or take advantage of Markdown for nicely formatted notes.

Code

Code documents make it easy to work on a wide range of syntaxes for different file types. More than a 100 different syntaxes are supported. Once a code document is open, you can view and set the document's syntax. A wide range of editor themes are also available for you to choose from.

Note

Note documents are nestable notes great for lists or organizing hierarchal data. Each note element can have children elements that can be shown or hidden. Elements can also be scoped to allowing you to easily create notes that are many levels deep. Traversing elements are made easy with a simple interface as well as keyboard shortcuts.

Premium

The free version of Vynote has certain limitations and is also supported by ads that occasionally popup in the notification bar or ad modal. Vynote Premium removes those limits, ads, and unlocks a whole new set of features and customization.

Features

  • No Advertisements
  • Remove Limitations
    • Unlimited Folders
    • Unlimited Documents
    • Unlimited Active Document Tabs
    • Unlimited Document Contributors
  • Document Contributor Permissions
    • Determine which users can write, update, or delete in your document
  • Client-Side Encryption
    • Optionally encrypt any document with client-side encryption.
    • If enabled, Vynote never sees your encryption key or your unencrypted messages
    • Documents can still be shared with others users if they have the encryption key
  • Import and Export Documents
  • In-Document Shortcuts
    • Shortcuts are a short bit of text that reference a large snippet of text
    • Simply define a shortcut in the Settings menu and then access in documents with ${shortcut_name}
  • Version Control
    • Versions save the document's content in its current form to the version
    • You can revert to a version's content at a later time
  • Dark Theme
    • A site-wide theme that's easy on the eyes in low-light environments
  • Customization
    • Colored folder and document icons
    • Many other available customizations can be found in the Settings menu
  • New features and customizations are always being added
